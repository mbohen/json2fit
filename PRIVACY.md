# Privacy

json2fit is designed as a local-first browser application.

## What is processed

The app processes Polar Flow export files selected by the user.

## Where processing happens

Processing happens locally in the user's browser.

## What is not uploaded

The app does not upload Polar Flow exports, GPS tracks, workout data, heart rate data or account data to a backend server.

## Accounts

The app does not require user accounts.

## Optional external links

The app may contain an optional Buy Me a Coffee link. Opening that link takes the user to an external website.

## Local storage and offline cache

The app may use browser storage for language/theme preferences, local beta counters and offline app assets. User-selected Polar Flow files and generated export files are not intentionally stored in browser storage by the app.

## Analytics

This app does not use external analytics.
