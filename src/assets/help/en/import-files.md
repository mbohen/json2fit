# How to upload files to the converter

The safest path is to upload the full ZIP downloaded from your Polar account. The app unpacks it locally and classifies JSON files in the browser.

1. Drag the ZIP onto the import area or use Choose ZIP.
2. You can also choose individual JSON files or a folder with JSON files.
3. Wait for scanning and classification to finish.
4. Check the classification report, warnings and Garmin-ready cards before export.

Data is processed in browser memory and is not sent to a server. A damaged ZIP, invalid JSON or file without workout data is shown as an issue to review.
