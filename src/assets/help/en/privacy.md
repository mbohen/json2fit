# Privacy and local processing

Everything runs directly in the browser. User files stay on the device and are not sent to a server.

1. Upload data locally in the browser.
2. Conversion, file checks, map preview and reports run on your device.
3. Download generated files to your computer.
4. Use Clear app data if you want to remove current session data and app storage.

Offline mode stores only files required to run the app and local help instructions. Polar account ZIP, workout JSON, GPS data, TCX/FIT and result reports are not automatically saved for offline mode.
