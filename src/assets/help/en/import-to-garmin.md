# How to import files to Garmin Connect

Garmin Connect import is manual: download a file from the app, then upload it in Garmin Connect.

1. In the app, download a single TCX/FIT file or a ZIP package with activities.
2. If you downloaded a ZIP, unpack it locally on your computer.
3. In Garmin Connect, choose manual activity import.
4. Upload FIT or TCX files in batches and check the import result.
5. Compare rejected activities with the Garmin-ready report.

Garmin Connect may change the sport type or reject an activity with missing data. Keep the original Polar account export so you can repeat conversion after parser fixes.
