# Jak wrzucić pliki do konwertera?

Najbezpieczniej wgrać cały ZIP pobrany z konta Polar. Aplikacja rozpakowuje go lokalnie i klasyfikuje pliki JSON.

1. Przeciągnij ZIP na obszar importu albo użyj przycisku Wybierz ZIP.
2. Możesz też wybrać pojedyncze pliki JSON lub katalog z plikami JSON.
3. Poczekaj na zakończenie skanowania i klasyfikacji.
4. Sprawdź raport klasyfikacji, ostrzeżenia i karty Garmin-ready przed eksportem.

Dane są przetwarzane w pamięci przeglądarki i nie są wysyłane na serwer. Uszkodzony ZIP, błędny JSON albo plik bez danych treningowych zostanie pokazany jako problem do sprawdzenia.
