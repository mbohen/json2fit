# Jak zaimportować pliki do Garmin Connect?

Import do Garmin Connect jest ręczny: pobierasz plik z aplikacji, a potem dodajesz go w Garmin Connect.

1. W aplikacji pobierz pojedynczy TCX/FIT albo pakiet ZIP z aktywnościami.
2. Jeśli pobrałeś ZIP, rozpakuj go lokalnie na komputerze.
3. W Garmin Connect wybierz ręczny import aktywności.
4. Dodaj pliki FIT albo TCX partiami i sprawdź wynik importu.
5. Porównaj odrzucone aktywności z raportem Garmin-ready.

Garmin Connect może zmienić typ sportu albo odrzucić aktywność z brakującymi danymi. Zachowaj oryginalny eksport z konta Polar, żeby móc ponowić konwersję po poprawkach parsera.
