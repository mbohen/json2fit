# Co z danymi snu i wellness?

Dane snu, aktywności dziennej i wellness są eksportowane jako lokalne raporty, nie jako obietnica importu do Garmin Connect.

1. Wgraj ZIP z konta Polar i poczekaj na analizę wellness/snu.
2. Sprawdź liczniki dni aktywności, nocy snu, Nightly Recharge i dziennego tętna.
3. Pobierz Wellness ZIP, jeśli chcesz lokalne archiwum CSV, JSON albo HTML.
4. Nie importuj tych raportów jako aktywności sportowych w Garmin Connect.

Garmin Connect nie zapewnia stabilnego publicznego importu historii snu z Polar Flow. Brakujące wartości pozostają puste albo trafiają do ostrzeżeń; aplikacja nie tworzy fałszywych zer.
