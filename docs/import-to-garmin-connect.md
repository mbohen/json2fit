# Import To Garmin Connect

1. Export TCX or FIT files from json2fit.
2. Open Garmin Connect in a browser.
3. Use Garmin Connect's manual import flow.
4. Select the generated TCX or FIT file.
5. Review activity type, start time, duration, distance, map and heart-rate data after import.

Garmin Connect can still reject or reclassify files even when local validation passes.
